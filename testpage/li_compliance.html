WHATWG 4.4.8 LI Element Compliance Test

The <li> element represents a list item inside <ol>, <ul>, or <menu>.

Test 1: Basic li in ol

Simple ordered list with li elements.

Expected: 1. 2. 3.

  1. First item
  2. Second item
  3. Third item

Test 2: Basic li in ul

Simple unordered list with li elements.

Expected: • • •

  • Apple
  • Banana
  • Cherry

Test 3: Basic li in menu

Menu with li elements (semantic ul).

Expected: • • •

  • Copy
  • Cut
  • Paste
    •

    Test 4: li with value attribute

    The value attribute overrides the ordinal number.

    Expected: 1. 3. 4. (value="3" on second item, next continues from 3)

    1. Item 1
    2. Item 3 (value="3")
    3. Item 4 (should continue from 3)

    Test 5: li value with reversed ol

    Combined reversed on ol and value on li.

    Expected: 10. 5. 4. (starts at 10, jumps to 5, then 4)

    1. Item 10
    2. Item 5 (value="5")
    3. Item 4 (continues down from 5)

    The value can be negative per spec.
